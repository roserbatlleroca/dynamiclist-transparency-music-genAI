# Live List: Transparency in Music-Generative AI 

This repository is linked to the article *[Transparency in Music-Generative AI]*(https://www.researchsquare.com/article/rs-3708077/v1). Given the rapid advancements in generative AI within the music domain, the goal of this repository is to provide a continuously updated, live literature review (from eldest to newest). It includes a categorised list of publications that reflects new research and developments of AI transparency for music generation, helping the community stay informed on the latest progress in this evolving field.
